Nossa Plataforma Educacional

A infraestrutura completa para o aprendizado prático e moderno.

Estrutura Técnica da Solução

Ambiente do Cliente (Instituição/Aluno)

Requisitos mínimos para garantir uma experiência fluida ao acessar a plataforma e rodar simulações locais.

Hardware Mínimo

  • CPU: Intel i5 / AMD Ryzen 5 (ou superior)
  • RAM: 16 GB
  • Armazenamento: SSD com 50 GB de espaço livre

Software Padrão

  • SO: Windows 10/11 ou Linux (Ubuntu, Fedora)
  • Navegador: Google Chrome ou Firefox
  • Ferramentas: Clientes SSH, RDP, VNC.

Infraestrutura da Plataforma (Nosso Backend)

Nosso data center é construído com tecnologias robustas e de ponta para entregar performance e disponibilidade.

Hardware e Virtualização

  • Cluster de servidores de alta performance
  • Hypervisors: Proxmox VE
  • Armazenamento: SAN/NAS de alta velocidade

Software de Base

  • Simulação: GNS3, EVE-NG
  • Automação: Ansible, Terraform, Git
  • Monitoramento: Prometheus, Grafana, Zabbix
  • Segurança: pfSense, Snort/Suricata, OpenVPN
  • Backup: Proxmox Backup Server

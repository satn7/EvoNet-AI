Projeto de Rede (UC05)

Detalhes da infraestrutura de rede que suporta a plataforma EvoNet AI.

Planta Lógica e Endereçamento

Topologia e Segmentação

  • Topologia: Estrela, com um core de rede robusto e segmentação lógica para performance e segurança.
  • Endereçamento IP: Privado (RFC 1918), utilizando a faixa 192.168.0.0/16.
  • Segmentação: VLANs para isolar o tráfego de diferentes turmas, laboratórios e serviços de gerenciamento.

Exemplo de Cálculo de Sub-rede

Laboratório 1 (100 hosts): Necessita de 7 bits para hosts (2⁷ = 128).

Rede: 192.168.10.0/25
Máscara: 255.255.255.128

Laboratório 2 (50 hosts): Necessita de 6 bits para hosts (2⁶ = 64).

Rede: 192.168.20.0/26
Máscara: 255.255.255.192

Mídia Física e Dispositivos

  • Cabeamento: Cat.6 para links de alta velocidade (servidores, switches core) e Cat.5e para pontos de acesso. Conectores RJ-45 padrão TIA/EIA-568B.
  • Switches: Gerenciáveis Layer 2/3 (Cisco, Mikrotik).
  • Roteador/Firewall: Appliance virtualizado (pfSense).
  • Access Points (Wi-Fi): Ubiquiti UniFi.
  • Rack: Padrão 42U com organização via Patch Panels.

Rede Wi-Fi

SSIDs e Segurança

  • EvoNet-Lab: Segurança WPA3-Enterprise (autenticação via RADIUS), para alunos e staff.
  • EvoNet-Guest: Rede para convidados com portal cativo e isolamento de cliente.
  • Configuração: Canais não sobrepostos (1, 6, 11 em 2.4GHz) e uso da frequência de 5GHz para performance.
